Formatando texto!

Negrito/Destaque

Adicionando negrito dentro de uma frase

A tag <strong> é usada para dar destaque dentro de um texto. Também existe a tag <b>, porém como o foco é manter o código com semâtica, usamos strong.

Italico/Ênfase

Adicionando Italico(ênfase) com a tag <i> é a forma não semântica.

Portanto, caso seja necessário usar a ênfase, usamos as tag <EM>

Texto marcado

É possível marcar texto utilizando a tag <mark>,além disso, podemos alterar a cor da marcação utilizando a tag style na tag caso seja uma alteração pontual, caso seja para todas tags mark, é melhor utilizar a tag style no head.

Texto grande e pequeno

É possível criar textos com letras pequenas, através das tags <small>, normalmente usada para letras no rodapé. A tag para aumentar as letras é obsoleta.

Texto deletado

Para adicionar significado a uma parte do texto que não deve ser considerada, utiliza-se a tag <del>

Texto inserido

Para sinalizar que o trecho do paragrafo foi inserido posteriormente, usa-se a tag <ins>

Texto sobrescrito

Util para colocar números em elevação, como x2- y2 (usa-se a tag <sup>)

Texto subscrito

Para colocar o texto mais abaixo, usamos a tag <sub>, como por exemplo H2O
