Negrito/Destaque

Adicionando negrito dentro de uma frase

A tag <strong> é usada para dar destaque dentro de um texto. Também existe a tag <b>, porém como o foco é manter o código com semâtica, usamos strong.

Italico/Ênfase

Adicionando Italico(ênfase) com a tag <i> é a forma não semântica.

Portanto, caso seja necessário usar a ênfase, usamos as tag <EM>
