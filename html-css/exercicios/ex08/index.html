Links externos

Aponta para links externos, como por exemplo: clique aqui para acessar meu Github

Pode também acessar a aula em que aprendi isso.

target="_blank" rel="external" são parametros usado após o href na tag <a>, servem para abrir uma nova aba ao acessar o link, assim a página do seu site é mantida.

Links Internos

Liga um link para uma pasta que está presente dentro do seu servidor. como por exemplo, acesse aqui a página de contato

Se o link for para a proxima pagina, utilizamos o parametro rel="next"

Existe o parametro rel="nofollow", quando você vai linkar uma pagina em que você não dá o seu aval.

Você pode conferir também a página de notícias clicando aqui

../ na hora de relacionar uma pasta significa que você quer um caminho da pasta anterior.
