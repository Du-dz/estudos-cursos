Formatações extras

Código-fonte e Pré-formatação

Utilizamos <code> quando precisamos colocar um código em um paragrafo, como: document.getElementById('teste').

Também é possível envelopar a tag <code> com a tag <pre>, para adicionar indentação (manter a forma). Exemplo:

        
    num = int(input('Digite um número'))
    if num % 2 == 0:
        print(f'O número {num} é PAR')
    else:
        print(f'O número {num} é ÍMPAR')
    print('Fim do programa')

A tag <pre> mantém a forma do texto igual no código e visualmente no site.
