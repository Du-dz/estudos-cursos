Listas Ordenadas são contidas dentro da tag <ol>, depois é necessario adicionar cada linha da lista dentro de uma tag <li>

É possível alterar o tipo de lista (romano/a,b,c/A,B,C) utilizando o parametro (type="") na tag ol

Listas não ordenadas são contidas em tags <ul>, também utilizando li para adicionar as linhas

Existem 3 tipos no ul (type=""), são eles disc(padrão), square e circle

Para criar uma lista de definições, colocamos entre a tag <dl>.

Dentro da tag dl, temos duas tags, <dt>, onde colocamos o termo, e <dd>, onde colocamos o significado do termo.

A ferramenta de pesquisa do Google da prioridade para paginas com listas de definições quando o assunto é procurar o significado de algum termo.
